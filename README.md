# DevHub | Professional Full-Stack & Backend Portfolio

A highly optimized, production-ready personal portfolio tailored for Software Engineers. This platform serves as a centralized hub to showcase complex programming accomplishments, spanning from low-level Unix system operations in C to robust, scalable backend infrastructures. It dynamically categorizes architectural workflows and presents formal engineering credentials in a clean, high-performance UI.

## ✨ Key Architectural Features

* **Domain-Based Architecture Filter:** Instead of a generic list, projects are segmented into dedicated tabs (`All`, `Full-Stack`, `Backend`, `Frontend`, `Systems`) to instantly direct technical recruiters to specific engineering domains.
* **Dual Theme Implementation:** Complete design system supporting high-contrast Dark and Light modes utilizing dynamic CSS Custom Variables (`--variables`) and structural transition matrices.
* **Verified Credentials Showcase:** A custom certification grid mapping out formal software engineering milestones, fundamental computer science topics, and analytical backgrounds with dynamic UI components.
* **Asynchronous Certificate Viewer:** Integrated interactive JavaScript modal (pop-up layer) to preview high-resolution credentials seamlessly, featuring dynamic body scroll-locking and multi-input closing event listeners (`ESC` key, backdrop click, Close button).
* **Direct Omnichannel Communication:** Optimized call-to-action (CTA) layout replacing traditional bulky forms with direct communication endpoints (WhatsApp API, Mailto email protocol, LinkedIn, and GitHub codebases).

## 📂 Portfolio Structure & Section Breakdown

The platform is engineered as a seamless single-page application (SPA) layout, navigated via a persistent sidebar control matrix:

1.  **Hero (Introduction) Layer:** Displays the primary core engineering statement, quick-access technical chat trigger, and professional subtitle.
2.  **About Me Narrative:** Details the unique transition from an analytical legal background to software engineering, demonstrating deep logical deductive skills.
3.  **Timeline Matrix:** Houses precise professional milestones, military service completion, and formal engineering training history.
4.  **Portfolio Grid:** The core repository display system, equipped with functional sorting filters for Backend, Frontend, and low-level Systems codebases.
5.  **Credentials & Certifications:** Dedicated space holding verified credentials for ALX Backend Specialization, Computer Science core logistics, and academic degrees.
6.  **Contact Hub (CTA):** A clean, form-less grid holding verified social gateways and production mail endpoints.

## 💻 Getting Started & Local Deployment

To host or inspect this responsive portfolio architecture locally, execute the following commands:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/mostafa-m-desouky/dev-hub.git](https://github.com/mostafa-m-desouky/dev-hub.git)
    ```

2.  **Navigate into the Directory:**
    ```bash
    cd dev-hub
    ```

3.  **Launch the Application:**
    Simply open the `index.html` file in any modern web browser, or serve it using a local development server extension (e.g., Live Server in VS Code).
